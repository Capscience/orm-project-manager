Instructions

1. Create user

First thing as a new user is to register a profile for the app. Profile menu can be found on right side of menu bar. After creating your user you will have to login. After that you have access to the app.

Why login?

Logging in is important for user safety and ease of use. It allows you to keep your projects and clients to yourself, but also makes using the app easier when you only have your personal projects and companies to manage.

2. Create a company

Before creating projects you must create a client company. In case you are using the app for schoolwork, you can think of this as the course. To create a new company, go to "App" and choose "New Company."

3. Create a project

Now projects can be created by choosing "New Project" from menu bar. You have different worktypes to use, but they can only be added by devs for now. Each worktype has different hourly price, minimum time, and rounding parameters when outputting billing data (not implemented).

4. Timing projects (yet to be implemented)

Projects will be timed by using start/pause and stop buttons. Start button will change into a pause button when pressed. Pausing does not round saved time at all, but stop will round the time according to selected worktype. You can click the expand button to view manual editing option for the project, which are also not implemented as of now.

General information

Currently little functionality is implemented, but development is going smoothly. Ready features have worked well in tests, and the base is pretty much ready for the rest of the app. Please report any issues to GitHub linked below. There you can also find all of the app's source code.
